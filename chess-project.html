Automated Chess Board

An innovative, engineering-focused build bringing automation to the traditional game of chess.

1. Introduction

The Automated Chess Board project is an innovative, engineering-focused build that brings automation into the traditional game of chess. Inspired by systems like commercial automated chess boards and top-tier electronic boards, the project aims to merge robotics, sensors, and software intelligence to create a fully automated board capable of recognizing moves, validating them, and physically moving chess pieces when required.

Developed as a university project, the system showcases teamwork, creativity, real-time problem‑solving, and technical implementation using both hardware and software components.

2. Project Concept & Vision

  • Detects piece movement automatically.
  • Identifies legal/illegal moves.
  • Physically moves pieces using an internal robotic mechanism.
  • Communicates with a computer or mobile app in real time.
  • Provides an interactive and elegant playing experience.

The vision was to replicate some features of commercial automatic boards while creating our own engineering logic, mechanical design, and software stack.

4. System Overview

4.1 Hardware

  • Chess board with 64 squares
  • Under‑board XY movement platform
  • Servo/stepper mechanism
  • Magnetic base pieces

4.2 Sensors

  • Hall‑effect sensors / reed switches
  • Real-time feedback
  • Magnetic detection

4.3 Software & Control

  • Arduino/ESP32/RPi
  • Move validation firmware
  • Path planning algorithms
  • Companion app

5. Working Principle

Step 1:
Move Detection - Sensors detect magnetic piece changes and generate input.
Step 2:
Move Verification - Logic checks legality and game state (check/checkmate).
Step 3:
Engine Interaction - Optional AI (Stockfish) generates counter-moves.
Step 4:
Automated Movement - XY mechanism moves electromagnet to drag piece to new square.

6. Mechanical Design

  • XY Plotter: CoreXY/belt-driven system with two stepper motors for diagonal free movement.
  • Magnetic Pickup: Electromagnet on actuator head to pick/drag pieces without collisions.
  • Board Layers: Top wood/acrylic grid, Sensor layer (8x8 Hall matrix), Movement layer, and Control layer.

7. Electronics & Utils

Microcontrollers: Arduino Mega / ESP32

Drivers: A4988 / DRV8825 for NEMA17 steppers

Power: 12V for motors, 5V for logic

8. Software Architecture

9. Algorithms

  • Move Detection: Comparison of current vs previous sensor matrix.
  • Path Planning: A* or direct path algorithm avoiding collisions.
  • AI Integration: Stockfish via UCI protocol.
